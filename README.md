# 「30日でできる! OS自作入門」 を一通り実践する

- 機種 : MacBook Pro (13-inch, 2017, Two Thunderbolt 3 ports)  
- プロセッサ : 2.5 GHz Intel Core i7  
- メモリ : 16 GB 2133 MHz LPDDR3
